# Tools

Overview of software, libraries, and CLI tools that generate, read, or convert FAIR-IO or its components.

* **NGFF-Converter**

* [**OME-NGFF-Validator**](https://ome.github.io/ome-ngff-validator/) validates OME-Zarr v.0.5 files containing ro-crate-metadata.json.

* [**omero-cli-transfer**](https://github.com/ome/omero-cli-transfer), which allows export of OMERO image data compliant with RO-Crate

* [**omero-rdf**](https://github.com/German-BioImaging/omero-rdf), allows you to export RDF from OMERO
